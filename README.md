# MediaPlanning

A web application to easily track advertising spend against a budget.  Event ticket sales are intended to be tracked to show income trends against advertising spend.

Currently uses:
 * Rails 4.1
 * Bootstrap 3


### Testing

Backed by rspec and cucumber tests.
